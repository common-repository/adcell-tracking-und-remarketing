=== ADCELL Tracking ===
Tags: tracking, retargeting
Requires at least: 4.6
Tested up to: 6.5.2
Stable tag: 4.6
Requires PHP: 7.0.0

This plugin adds Tracking- and Retargeting-Code for the ADCELL-Network to WooCommerce shop.

== Description ==

About ADCELL
Founded by Oliver and Marcus Seidel in 2003, ADCELL has meanwhile become the market leading affiliate network for small and medium enterprises in the DACH area.
We strive to deliver the best product in the industry and support our clients with outstanding customer service and tailor made solutions.
More than 1800 advertisers and more than 145000 publishers benefit from our state of the art technology and our advanced platform.

Plug-in overview
The Plug-in enables you implementing the ADCELL conversion tracking and our container tags for your WooCommerce online shop easily.
For using the Wordpress Plug-in it is necessary to be registered as advertiser in our network. ADCELL charges neither setup, nor monthly fees.
Please sign up here.
Plug-in Description

The Plug-in enables you for the following tracking methods:
• First Party Tracking
• Cookie Tracking
• Session Tracking
• Container Tag Implementierung
Our dashboard gives you an transparent overview and provides you with all the relevant KPIs.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/roleWcAdcellTracking` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->ADCELL Tracking screen to configure the plugin
